# IanLangs

IanLangs es un ecosistema de lenguajes y herramientas experimentales.  
Lenguajes reales, tooling real, sin humo.

El foco está en:
- Transpilación
- Lenguajes de bajo a alto nivel
- Compatibilidad y bajo friction
- Experimentación seria
- **icc**: compilación de lenguajes serios (no experimentales)

## Proyectos

- **Flow-C++**  
  Superset de C++ con mejoras de flujo, tipos y seguridad, transpilado a C++ estándar.

- **BF# (BrainFuckSharp)**  
  Extensión moderna de Brainfuck con stack, strings, IO y control estructurado.
